## 📝 Entrega: Práctica 2 — Tipos, Variables y Control de Flujo

### 🎯 Objetivos de la Práctica
* Distinguir entre `val` y `var`, y aplicar inferencia de tipos.
* Usar los tipos básicos de Kotlin: `Int`, `Double`, `String`, `Boolean`.
* Aplicar `if/else` como expresión, `when`, `for` y `while`.
* Usar plantillas de cadena (string templates) con `$variable` y `${expresion}`.

### 🛠️ Soluciones Implementadas
1. **Ej. 1 (Variables y Templates):** Resueltos los TODO 1, 2 y 3 utilizando variables mutables e inmutables, e imprimiendo el formato solicitado con String Templates.
2. **Ej. 2 (Clasificador de Notas):** * ⭐ **Reto Adicional Completado (+0.5 pts):** Se modificó la función `clasificarNota()` para que acepte parámetros de tipo `Double`, permitiendo evaluar calificaciones con decimales de manera precisa.
   * Se añadieron las 6 pruebas requeridas más un caso de borde con nota inválida.
3. **Ej. 3 (FizzBuzz y Tabla):** * Parte A ejecutada imprimiendo del 1 al 30 con saltos de línea cada 10 elementos.
   * Parte B integrada con `readLine()?.toIntOrNull()` para permitir la interacción dinámica del usuario a través de la consola.
